My Profile

Next.js & Tailwind Editorial Portfolio

Architecting a minimalist narrative through code.

My Profile is a bespoke digital experience designed to challenge the traditional grid-heavy portfolio archetype. The mission was to create a "silent" interface—one that prioritizes typography and breathing room over decorative clutter.

Built to serve as a modular framework for creative professionals, it solves the problem of visual exhaustion in developer showcases by treating web content with the same reverence as high-end print journalism.
